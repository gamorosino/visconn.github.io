VISion CONNectivity toolbox

Retinotopic connectivity for the human visual system.

VISCONN integrates retinotopic maps with diffusion MRI tractography to estimate structural connectivity between cortical locations defined in visual-field coordinates.

View repository Read preprint Browse Apps

VISCONN links cortical maps of eccentricity and polar angle to white-matter pathways, allowing connectivity to be analyzed in the coordinate system of visual space.

Overview

The human visual cortex contains aligned retinotopic maps across multiple areas. VISCONN was developed to test whether this representational geometry is also reflected in structural connectivity. The toolbox reconstructs visual pathways as a function of eccentricity and polar angle, producing retinotopic connectivity matrices and tract profiles.

What VISCONN reveals

Like-to-like wiring

Structural connectivity is strongest between cortical regions that represent homologous locations in visual space.

Perceptual asymmetries

Connectivity patterns mirror known horizontal–vertical and vertical-meridian asymmetries in human visual performance.

Population templates

Aggregated tractography can recover retinotopic pathways that are difficult to reconstruct in individual brains.

Workflow

VISCONN combines structural MRI, diffusion MRI, anatomically predicted retinotopic maps, and tractography. Streamline endpoints are assigned to retinotopic parcels, enabling connectivity estimates between cortical locations that represent corresponding positions in the visual field.

Inputs

T1-weighted MRI, diffusion-weighted MRI, cortical surfaces, retinotopic maps, and whole-brain tractography.

Outputs

Retinotopic connectivity matrices, retinotopic tract profiles, streamline-density measures, and population-level visual connectivity templates.

Installation

The VISCONN workflow is distributed as versioned Apps on brainlife.io. If using a local repository, clone the project and install it in editable mode. 

git clone https://github.com/gamorosino/<\repository-name\>.git
cd repository-name
pip install -e .
